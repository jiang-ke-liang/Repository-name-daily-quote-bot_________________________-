 # Daily Quote Bot 🎯

&gt; 一个基于 Python 的命令行名言管理系统，支持每日随机名言、分类查询、增删改查功能。

## 功能特性

- ✅ 每日随机名言（基于日期种子，每天固定）
- ✅ 按分类筛选（技术/人生/幽默/测试）
- ✅ 添加新名言（带输入验证）
- ✅ 删除名言（编号删除）
- ✅ 修改名言（可选字段）
- ✅ 数据持久化（JSON 文件存储）

## 技术栈

- Python 3.13
- 面向对象编程（OOP）
- JSON 文件操作
- 异常处理

## 项目结构
Repository-name-daily-quote-bot/
├── quote_manager.py    # 核心模块（Quote + QuoteManager 类）
├── main.py             # 主程序（用户交互菜单）
├── quotes.json         # 数据文件
└── README.md           # 项目说明
